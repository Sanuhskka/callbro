# Быстрый Старт
# Secure P2P Messenger

Это руководство поможет вам запустить Secure P2P Messenger локально за 5 минут.

## Предварительные Требования

- Docker и Docker Compose установлены
- Git установлен
- Порты 80, 8081, 3478, 5432 свободны

## Шаги

### 1. Клонировать Репозиторий

```bash
git clone https://github.com/your-org/secure-p2p-messenger.git
cd secure-p2p-messenger
```

### 2. Создать Файл Окружения

```bash
cp .env.example .env
```

Для локального тестирования значения по умолчанию подходят.

### 3. Запустить Приложение

```bash
docker-compose up -d
```

Это запустит:
- PostgreSQL базу данных
- WebSocket сигнальный сервер
- TURN/STUN сервер (Coturn)
- React веб-клиент

### 4. Проверить Статус

```bash
docker-compose ps
```

Все сервисы должны быть в состоянии "Up".

### 5. Открыть Приложение

Откройте браузер и перейдите на:
```
http://localhost
```

### 6. Создать Учетную Запись

1. Нажмите "Регистрация"
2. Введите username и пароль
3. Нажмите "Зарегистрироваться"

### 7. Тестирование P2P Соединения

Для тестирования P2P функциональности:

1. Откройте приложение в двух разных браузерах (или в обычном и приватном режиме)
2. Зарегистрируйте два разных аккаунта
3. Добавьте друг друга в контакты
4. Отправьте сообщение или совершите звонок

## Просмотр Логов

```bash
# Все сервисы
docker-compose logs -f

# Только сервер
docker-compose logs -f server

# Только клиент
docker-compose logs -f client
```

## Остановка Приложения

```bash
docker-compose down
```

## Удаление Всех Данных

```bash
docker-compose down -v
```

**ВНИМАНИЕ**: Это удалит все данные базы данных!

## Устранение Неполадок

### Порт уже используется

Если порт 80 занят:
```bash
# Изменить порт в .env
CLIENT_PORT=8080

# Перезапустить
docker-compose down
docker-compose up -d
```

### Контейнеры не запускаются

```bash
# Проверить логи
docker-compose logs

# Пересоздать контейнеры
docker-compose down
docker-compose up -d --force-recreate
```

### База данных не инициализируется

```bash
# Удалить volume и пересоздать
docker-compose down -v
docker-compose up -d
```
